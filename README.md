# 🚀 Swift Login & Signup

> **My first SwiftUI project and the beginning of my iOS development journey.**

This repository represents the first milestone in my journey of learning Swift and SwiftUI. Coming from a web development background, I wanted to explore Apple's ecosystem and start building native applications for iPhone, iPad, and Mac.

Although this is a beginner project, every line of code written here has helped me understand the fundamentals of SwiftUI and modern Apple app development.

---

## 📱 Project Overview

This project contains a clean and modern Login & Signup interface built entirely using **SwiftUI**.

It focuses on learning:

- SwiftUI layouts
- Navigation between screens
- Reusable UI components
- TextFields & SecureFields
- Buttons and styling
- Images & Assets
- Modern Apple UI design principles

---

## ✨ Features

- 🔐 Login Screen
- 📝 Sign Up Screen
